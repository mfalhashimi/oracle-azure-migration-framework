# Oracle to Azure Migration Checklist

## 1. Assessment Phase

- Inventory Oracle databases
- Review Oracle RAC and Data Guard topology
- Identify application dependencies
- Review security and compliance requirements
- Validate Oracle licensing considerations

---

## 2. Migration Planning

- Define migration strategy
- Define RPO/RTO requirements
- Prepare rollback procedures
- Define cutover approach
- Review network connectivity requirements

---

## 3. Azure Environment Preparation

- Deploy Azure SQL Managed Instance
- Configure VPN / ExpressRoute
- Configure Azure Monitor
- Configure Microsoft Purview
- Configure backup and DR policies

---

## 4. Migration Execution

- Initial schema migration
- Data synchronization
- Incremental replication
- Data validation testing
- Performance testing

---

## 5. Cutover

- Freeze source changes
- Final synchronization
- Redirect applications
- Validate application connectivity
- Monitor workload stability

---

## 6. Post-Migration

- Validate backups
- Review security posture
- Optimize SQL performance
- Enable governance workflows
- Review monitoring dashboards
